1. Version 0.2 Wrap-Up

Version v0.2 was completed at the end of January 2025. We didn't post a formal announcement at the time since we immediately moved on to v0.3 development.

We spent an inordinate amount of time on the v0.2 data model. By refusing to add more features until it became painful, we were forced to stretch the model as far as it could go.

The declarative model now includes improved data structures, expression broadcasting, indexing and slicing, and richer operators across matrix and set data.

Of course there are strict limitations on the kinds of programs you can write with that set of features, so today we are launching Mech v0.3 with a whole new set of control-flow mechanisms that make Mech more capable.

2. Enums, Tagged Unions, and Pattern Matching

Mech now has an enum and tagged-union system:

enum.m
<color> ::= red | green | blue
<xcolor> ::= red
Error: GenericError

Enums and tagged unions allow for modeling categorical data and optional values with type safety and clarity.
