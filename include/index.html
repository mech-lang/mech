1. Version 0.2 Wrap-Up

Version v0.2 was completed at the end of January 2025. We spent significant effort on improving the data model so new control-flow primitives could be introduced in v0.3.

The declarative model now includes improved data structures, expression broadcasting, indexing/slicing mechanics, and richer operators across matrix and set data.

Of course there are strict limitations on the kinds of programs you can write with those features, so today we are launching Mech v0.3 with a whole new set of control-flow mechanisms that make Mech more capable.

2. Enums, Tagged Unions, and Pattern Matching

Mech now has an enum and tagged-union system:

enum.m
<color> ::= red | green | blue
<xcolor> ::= red
Error: GenericError

Enums and tagged unions model categorical data and optional values with strong type safety and clarity.
